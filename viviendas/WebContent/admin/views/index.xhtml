Gestión de Matriculas

Sistema Viviendas

Este módulo del sistema gestiona la parte interna o control total de los recursos para que el alumno pueda realizar una correcta reservación de habitacion.El cual consta de una gestión en la creación de periodos académicos, Una fácil asignación de sitios para poder ser reservados por género y Carga de matriculados para cada uno de los periodos asignados.
